
## Переменные

Переменная - название ячейки памяти, в которой хранится некоторое значение определеного типа. 

```python
a = 2 + 2
a
```

```python
a = 2 + 2
print(a)
```

```python
a = 2 + 2
print(f"{a}")
```

```python
a = 2
b = a + 2
c = b
print(f"{a} {b} {c}")
```
